How to Use the Trade Calculator

Adding Vehicles

Select vehicles from the dropdown menus and click "Add Vehicle" to add them to each side of the trade. You can add the same vehicle multiple times.

Calculating Trades

Once you've added all vehicles to both sides, click "Calculate Trade" to see if the trade is a win, fair, or loss.

Understanding Results

The calculator will determine the outcome based on the following criteria:

  • WIN: You're receiving more value than you're giving
  • FAIR: The value difference is 500 or less on either side
  • LOSS: You're giving more value than you're receiving
